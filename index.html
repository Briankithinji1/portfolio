Law Firm Api

I leverage Java, Spring Boot, and PostgreSQL to create a resilient backend, serving as the cornerstone for a comprehensive law firm management solution. A critical aspect is secure file storage, including legal documents. To ensure scalability and security, I seamlessly integrate GCP (Google Cloud Platform) for both hosting and file storage using Cloud Storage. With the flexibility of Spring Boot, streamlined deployment through Docker, and the robust capabilities of GCP, the Law Firm Management System API emerges as an efficient solution, combining modern technologies for seamless law firm management.

Pharmacy Api

I utilize Java and Spring Boot to build a robust backend. This API acts as the central hub for a pharmacy management system, facilitating seamless communication. Key to the project is the secure storage of files such as images and prescriptions. Leveraging AWS S3, a reliable object storage service, ensures efficient and secure file handling. With the agility of Spring Boot and streamlined deployment through Docker, the Pharmacy API stands as a powerful solution, combining modern technologies for effective pharmacy management.
